= omicron-debug-dropbox

The Debug Dropbox is a well-known filesystem path into which parts of https://github.com/oxidecomputer/omicron[Omicron] (the Oxide control plane) can deposit small bits of debug data that will be archived for possible future use by support.  See https://rfd.shared.oxide.computer/rfd/613[RFD 613 Debug Dropbox] for background.

This repo contains a crate that's used for dropping debug data into the dropbox.  See the crate documentation for details.
